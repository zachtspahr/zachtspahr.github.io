Basic Description of This Site

This is the home page with some basic data on the congressional districts in Virginia. I am hoping to generally build up this test site/ make it look better, but I have included a few graphs to see if I can highlight a few trends in the districts that flipped in the state. This homework submission might be rough, but I hope that it will gradually get better over time. Here are some graphs with some images. These graphs were done in excel because it was easier to graph data over time in excel rather than python.

This is the data from all the congressional districts in VA from the 2008, 2012, 2016, presidential elections. The goal is to look at the data and see if different types of districts are trending in different ways. Also, I hope to look later at how demogrpahics may be playing a role in voter preferences.
